🧩 Actividades

Actividad 1: Comprensión de la matriz 4x4x4
 - Representación física y visual de una matriz tridimensional.
 - Explicación de coordenadas (X, Y, Z) aplicadas a LEDs.

 Actividad 2: Simulación de una capa del cubo LED en Tinkercad
 - Uso de 4 LEDs conectados en forma de matriz 2x2.
 - Programación básica con bloques o código para encender los LEDs según posiciones.

 Actividad 3: Exploración del código Arduino (estructura básica)
 - Presentar un código básico y explicar su estructura:
   void setup() – se ejecuta una sola vez al iniciar.
   void loop() – se repite constantemente, contiene el comportamiento principal.
 - Actividad práctica: modificar el tiempo de encendido y observar el efecto.

 Ejemplo de código:
 void setup() {
   pinMode(13, OUTPUT);
 }

 void loop() {
   digitalWrite(13, HIGH);
   delay(1000);
   digitalWrite(13, LOW);
   delay(1000);
 }
